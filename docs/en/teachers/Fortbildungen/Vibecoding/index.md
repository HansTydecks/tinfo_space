# Vibecoding in Teaching

Vibecoding enables teachers without programming experience to create apps and websites that cover a variety of use cases. From classroom management apps to content delivery, the [possibilities](../../Digitale_Tools/index) are endless!

Therefore, the **goal of this training**: teachers can fully implement a computer simulation by using an AI agent to write the program.

## Setup

To vibecode as efficiently as possible, it makes sense to integrate LLMs directly into your IDE of choice. Even if these terms don't mean anything to you yet, you can follow this tutorial to start programming in just a few clicks. Whether novice or expert, the guide explains every step in as much detail as possible.

[Continue reading](setup.md)
