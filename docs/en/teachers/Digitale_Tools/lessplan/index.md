# Lesson Plan Creator (fast-lessonplan)

Live: https://lessplan.tinfo.space/
Repo: https://github.com/HansTydecks/fast-lessonplan

Brief description

A lightweight, static browser tool for quickly creating lesson plans. The tool allows creating phases, automatic time calculation, learning objectives, sequencing of social forms and PDF export of the finished lesson.

Didactic notes

- Use scenarios: Planning individual lessons, quick preparation of substitute lessons or as a template for teaching series.
- Teaching activities: Record time allocation, group assignments and material lists directly in the plan; PDF export for filing or sharing with colleagues.
- Differentiation: Use the learning objectives section to note tasks at different difficulty levels; export/import (JSON) allows exchange between teachers.

Technical notes

- Purely client-side (HTML/CSS/JS), uses jsPDF for PDF export and saves plans as JSON for backup and exchange.
- No login required, can be operated locally or as a GitHub Pages site.
