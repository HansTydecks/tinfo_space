Live: https://memory.tinfo.space/
Repo: https://github.com/HansTydecks/memory-visualizer

This simulation is used in combination with storage media as demonstration objects. The storage media are in the classroom. Learners check on the objects themselves how large they are. They find the files on their computer and check with the file explorer how large they are. The simulation ultimately puts the media in relation and demonstrates how many files fit on a medium.

Required:
- Text file (e.g. .DOCX), music file (.mp3), pixel graphic (.jpeg)
- physical: CD, DVD, hard drive, USB stick with label (size), mobile phone (have learners check themselves in settings), floppy disk

Notes: the operation is a bit special. Try it yourself and give suggestions for improvement.
