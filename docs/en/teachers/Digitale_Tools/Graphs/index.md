# Graph Theory and Network Topologies

- [Live demo](https://graphs.tinfo.space/)
- [GitHub repository](https://github.com/HansTydecks/network-topology)

Learners can add nodes and connect them with edges. This helps them discover different topologies and observe data flow at the touch of a button.

## Features

- BFS and DFS simulation
- Failure simulation
- Multiple analysis tools
- Topology explanations after exploration

## Classroom use

The simulation works well as an introduction to graph theory and networks. Teachers can create supporting materials to adapt tasks to their lessons.
