# Logic Stories (Lories)

- [Live demo](https://lories.tinfo.space/)
- [GitHub repository](https://github.com/HansTydecks/logic-story)

Three different stories can be completed with little or no programming experience.

## Features

- Three interactive stories
- Focus on programming logic and variables
- Suitable for introductions and lesson starters
- Dyslexia support for text-heavy passages

## Classroom use

The tool is a low-threshold way to introduce algorithmic thinking and first programming concepts.
