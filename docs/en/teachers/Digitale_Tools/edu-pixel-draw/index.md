# Pixel & Bits (Edu Pixel Draw)

Live: https://pixel.tinfo.space/
Repo: https://github.com/HansTydecks/edu-pixel-draw

Brief description

A pixel grid that shows how bits are used for image representation; includes missions to progressively unlock more colors.

Didactic notes

- Use: Introduction to bits & colors, visualization of bit depth and storage space.
- Activities: Missions (e.g. draw a heart) and comparisons of different color depths.
- Technology: Runs completely client-side; ideal for stationary and mobile devices.
