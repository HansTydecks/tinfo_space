# Word Reveal

Live: https://words.tinfo.space/
Repo: https://github.com/HansTydecks/word-reveal

Brief description

Interactive gap text generator: enter texts, select words, use gap text.

Didactic notes

- Use: Gap texts for grammar exercises, vocabulary tests or reading comprehension.
- Differentiation: Choose words of different difficulty; use larger font or projector mode for presentations.
- Privacy: No user accounts — everything runs locally in the browser.
