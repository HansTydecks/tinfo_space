# Dilemmataxi - a "survey" on autonomous driving

Live: https://dilemmataxi.tinfo.space/
Repo: https://github.com/HansTydecks/car-dilemma

Brief description

Learners should answer a variety of questions about autonomous vehicles.

Didactic notes

- Use: Discussion of machine decision-making processes.
- Activities: plan about 40 minutes for learners to complete the survey. There will be giggling, but also discussion. Afterwards, one could start a discussion "utilitarianism vs. principle-based ethics".
