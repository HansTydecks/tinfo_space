# Binary-ASCII-Visualizer

Live: https://bascii.tinfo.space/
Repo: https://github.com/HansTydecks/binary-ascii-visualizer

Brief description

Visualizes 8 bits as switches, ASCII characters, decimal number and grayscale display.

Didactic notes

- Use: Demonstration of how the same bits can have different meanings (ASCII, decimal, image).
- Activities: Bit switch experiments, conversion tasks, comparisons between representation forms.
