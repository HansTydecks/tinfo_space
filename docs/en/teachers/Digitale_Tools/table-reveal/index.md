# Table Reveal

Live: https://tabula.tinfo.space/
Repo: https://github.com/HansTydecks/table-reveal

Brief description

A tool that reveals tables step by step (CSV/Excel/TSV) — with timer, theme options and responsive design.

Didactic notes

- Use: Exam rounds, interactive practice phases, vocabulary presentation in table form.
- Activities: Time-controlled revelation, collect student answers, grouping of data for discussion.
