# Card Creator

Live: https://ccreator.tinfo.space/
Repo: https://github.com/HansTydecks/card-creator-website

Brief description

A simple, static tool for creating and exporting flashcards as PDF.

Didactic notes

- Use: Production of individual learning cards for vocabulary training, class pools or assessment cards.
- Customization: Teachers can create templates (e.g. image on front, solution on back).
- Differentiation: Use smaller card sets for weaker learning groups and larger sets for advanced learners.
