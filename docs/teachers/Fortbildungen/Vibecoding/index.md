# Vibecoding im Lehreralltag

Vibecoding ermöglicht es Lehrkräften ohne Programmiererfahrung Apps und Webseiten zu erstellen, die eine Vielzahl von Use Cases abdecken. Von Apps zur Klassenzimmerorganisation bis hin zur Inhaltsvermittlung, die [Möglichkeiten](../../Digitale_Tools/index) sind endlos!

Daher das **Ziel dieser Fortbildung**: die Lehrkraft kann eine Computersimulation vollständig implementieren, indem sie einen KI-Agenten zum Schreiben des Programms nutzt.

## Setup

Um möglichst effizient zu vibecoden, bietet sich eine Integration der LLMs direkt in die IDE der Wahl an. Auch wenn dir diese Begriffe noch nichts sagen, kannst Du dem folgenden Tutorial folgen, um in wenigen Klicks mit dem Programmieren zu starten. Egal ob Noviz_in oder Expert_in, die Anleitung erklärt jeden Schritt möglichst detailliert.

[Weiterlesen](setup.md)

## How-to vibecode: praktische Anleitung ohne Vorwissen

Egal ob langjährige Informatiklehrkraft mit Programmiervorgeschichte oder neugierige Lehrkraft ohne jede Erfahrung mit dem Programmieren: jede und jeder schreibt nach wenigen Minuten individuelle Apps, die den Unterricht interaktiver und spannender gestalten. Klicke auf den Link, um eine Schritt-für-Schritt-Anleitung zu erhalten, wie Du am geschicktesten vorgehst.
